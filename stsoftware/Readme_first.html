Introduction

STM32CubeMonitor-Power is a graphical tool :

  • Allowing end user to display on PC power data coming from PowerShield HW Board (connected on ST or competitors eval boards):
    • With high accuracy power data values (from 100 nA to 10 mA)
    • At high sampling rate (up to 100 kHz)
  • Bringing advanced features to analyze data (zoom, measurements reports, etc.).

Another important feature of STM32CubeMonitor-Power is to perform ULPBench tests and compute an ULPMark score estimation.

Installing and running STM32CubeMonitor-Power

Supported operating systems and architectures

  • Windows® 7 or newer: 32-bit (x86), 64-bit (x64)
  • Linux® (tested on RedHat, Fedora and Ubuntu, 32 and 64 bits)
  • macOS®: minimum version OS X® Yosemite or newer

Software requirements

The Java Run Time Environment 1.8 (version 1.8.121 or newer) by Oracle® must be installed.

WARNING:

  • Java OpenJDK is not supported.
  • Java™ SE Run Time Environment 9 by Oracle® is not supported.

Installing STM32 USB Virtual COM Port Driver STSW-STM32102

STSW-STM32102 driver must be installed to allow STM32CubeMonitor-Power to connect to X-NUCLEO-LPM01A.

STSW-STM32102 driver can be downloaded from here.

To install STSW-STM32102 driver, launch the executable in the zip file corresponding to the Windows® version and the CPU architecture (32-bit (x86) or 64-bit (x64)).

Installing STM32CubeMonitor-Power

If another version of STM32CubeMonitor-Power is already installed, the existing version should be uninstalled before installing the new version (see Uninstalling STM32CubeMonitor-Power to uninstall the current version).

To install STM32CubeMonitor-power, follow the steps below:

Extract (unzip) the whole content of the the latest STM32CubeMonitor-Power installation package from http://www.st.com into the same directory.

  • On Windows
    • Make sure you have administrators rights
    • Double click on the SetupSTM32CubeMonitor-Power-X.Y.Z.exe file
  • On Linux
    • Make sure you have access rights to the target installation directory
    • Double click on the SetupSTM32CubeMonitor-Power-X.Y.Z.jar file
    • Or launch manually the installation with java -jar <install path>/SetupSTM32CubeMonitor-Power-X.Y.Z.jar
  • On MacOS
    • Make sure you have access rights to the target installation directory
    • Double click on the SetupSTM32CubeMonitor-Power-X.Y.Z.dmg file
    • Open the STM32CubeMonitor-Power new disk
    • Drag and drop STM32CubeMonitor-Power shortcut to the Applications shortcut

Note: X.Y.Z represents STM32CubeMonitor-Power SW version.

After the installation you can safely remove the content of the zip from your disk.

Please refer to the STM32CubeMonitor-Power user manual for more details on System requirements or other possible installations.

Running STM32CubeMonitor-Power

  • On Windows
    • Select STM32CubeMonitor-Power from Program Files > ST Microelectronics > STM32CubeMonitor-Power (or from the path selected during installation).
    • Or double-click STM32CubeMonitor-Power icon on your desktop
  • On Linux
    • Launch the STM32CubeMonitor-Power executable from the STM32CubeMonitor-Power installation directory
    • Or double-click STM32CubeMonitor-Power icon on your desktop
  • On MacOS
    • Launch the STM32CubeMonitor-Power executable from the MacOS Applications directory
    • Or double-click STM32CubeMonitor-Power icon from MacOS Applications shortcut on your desktop

Uninstalling STM32CubeMonitor-Power

To uninstall STM32CubeMonitor-Power, follow the steps below:

  • On Windows
    • Open the Windows Control panel.
    • Select Programs and Features to display the list of programs installed on your computer.
    • Right click on STM32CubeMonitor-Power and select the uninstall function
  • On Linux
    • Launch uninstaller.jar in the STM32CubeMonitor-Power > Uninstaller directory
    • Tick "Force the deletion" and press Uninstall button
  • On MacOS
    • Directly move the STM32CubeMonitor-Power application in the recycle bin
